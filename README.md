🏦 Loan Approval Prediction Using Machine Learning

📌 Project Overview

This project aims to predict whether a loan application will be approved or rejected based on applicant details such as income, CIBIL score, loan amount, education, employment status, and asset values. The project uses Machine Learning algorithms to help financial institutions make faster and more accurate loan approval decisions.

🎯 Objective

To build a Machine Learning model that predicts loan approval status using applicant financial and personal information.

🛠️ Technologies Used

Python,
Pandas,
NumPy,
Matplotlib,
Seaborn,
Scikit-learn,
Jupyter Notebook

📂 Dataset Features

Number of Dependents,
Education,
Self Employed,
Annual Income,
Loan Amount,
Loan Term,
CIBIL Score,
Residential Asset Value,
Commercial Asset Value,
Luxury Asset Value,
Bank Asset Value,
Loan Status (Target Variable)

⚙️ Project Workflow

Data Collection and Understanding,
Data Cleaning and Preprocessing,
Exploratory Data Analysis (EDA),
Feature Encoding,
Train-Test Split,
Model Building,
Model Evaluation,
Loan Status Prediction

🤖 Machine Learning Models Used

Logistic Regression,
Decision Tree Classifier,
Random Forest Classifier

📊 Results

Model	Accuracy
Logistic Regression	81%
Decision Tree	97%
Random Forest	98.88%

Random Forest achieved the highest accuracy and was selected as the final model.

🔍 Key Insights

CIBIL Score is the most important factor affecting loan approval.
Higher income and asset values increase approval chances.
Loan amount and loan term significantly influence decisions.
Random Forest effectively captures complex relationships in the data.
✅ Conclusion

I developed a Loan Approval Prediction System using Machine Learning techniques. After preprocessing, analysis, and model comparison, Random Forest achieved the best accuracy of 98.88%. This project demonstrates how Machine Learning can support faster and more accurate loan approval decisions.
